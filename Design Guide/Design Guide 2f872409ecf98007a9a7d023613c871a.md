## **DETAILED GAME MECHANICS**

### **TACTICAL COMBAT:**

**Board:**

- 10Ã—10 grid (640Ã—640 pixels, 64px tiles)
- Terrain types affect combat:
    - Plains: Neutral
    - Forest: +5 DEF, provides cover from ranged
    - Mountain: +3 DEF, +1 range for archers, costs 2 movement
    - Water: -2 DEF, costs 2 movement
    - Lava: 5 damage per turn

**Unit Stats:**

- HP (Hit Points): 30-120
- ATK (Attack): 10-30
- DEF (Defense): 5-20
- SPD (Speed): 5-12 (determines turn order)
- Movement Range: 1-2 tiles
- Attack Range: 1-3 tiles

**Combat Formula:**

gdscript

`base_damage = attacker.ATK - defender.DEF
terrain_atk_bonus = attacker's terrain.attack_bonus
terrain_def_bonus = defender's terrain.defense_bonus
modified_damage = max(1, base_damage + terrain_atk_bonus - terrain_def_bonus)

# 10% crit chance (2Ã— damage)
if randf() < 0.1:
    final_damage = modified_damage * 2
else:
    final_damage = modified_damage
```

**Turn System:**
- Player turn â†’ Enemy turn â†’ Repeat
- Player controls all units during their turn
- Enemy AI acts during enemy turn
- Each unit can move once and attack once per turn
- Press Enter or click "End Turn" to switch

**Movement Rules:**
- Hero: 2 tiles in any direction (king-like)
- Elite Warrior: 2 tiles orthogonally or diagonally
- Elite Archer: 2 tiles movement, 3 tile attack range
- Pawn: 1 tile forward, diagonal to attack

**Win Conditions:**
- Eliminate all enemy units
- OR defeat enemy Hero unit
- (Boss battles may have special objectives)

### **ARMY MANAGEMENT:**

**Squad System (Banner Saga-inspired):**

**Army Structure:**
```

â”œâ”€ Active Squad (8 units) â† Deployed in battles
â”œâ”€ Reserve (unlimited) â† Backup units
â””â”€ Wounded (recovering) â† Unavailable for 1-2 battles
```

**Deployment:**
- Before each battle: Choose which 8 units to deploy
- Can swap units from Reserve into Active Squad
- Wounded units cannot be deployed
- Auto-deploy option available (AI picks best 8)

**Army Growth:**
```
Run Start: 5 units (1 Hero, 2 Elite, 2 Pawns)
Region 1 End: 10-12 units
Region 2 End: 18-20 units
Region 3 End: 25-30 units`

**Wounded System:**

- Unit reaches 0 HP â†’ Wounded (not dead)
- Wounded units unavailable for 1-2 battles
- Can pay gold at shops to heal immediately
- Auto-heal after completing a region

**Why This Works:**

- Battles stay 8v8 or smaller (manageable)
- Army size still matters (provides options and reserves)
- Wounded system creates tension (can't use same squad every battle)
- Encourages unit diversity
